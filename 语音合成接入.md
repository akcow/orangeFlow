# 语音合成-通义千问
更新时间：2025-12-01 16:54:13

## 产品详情
我的收藏

语音合成-通义千问提供多种拟人音色，支持多语言及方言，并可在同一音色下输出多语言内容。系统可自适应语气，流畅处理复杂文本。

## 支持的模型
推荐使用通义千问3-TTS-Flash。

- 通义千问3-TTS-Flash拥有 49种音色，支持多种语言及方言。
- 通义千问-TTS最多拥有7种音色，且只支持中文和英文。

## 计费规则
按输入的字符数计费，计算规则如下：
- 一个汉字（包括简/繁体汉字、日文汉字和韩文汉字）= 2个字符
- 其他（如一个英文字母、一个标点符号、一个空格）= 1个字符

## 快速开始

### 准备工作
1. 已配置 API Key 并配置到环境变量。
2. 若通过 DashScope SDK 调用：
   - DashScope Java SDK 版本 ≥ 2.21.9
   - DashScope Python SDK 版本 ≥ 1.24.6

### 说明
DashScope Python SDK 中的 `SpeechSynthesizer` 接口已统一为 `MultiModalConversation`，使用新接口只需替换接口名称，其他参数完全兼容。

### 关键参数
- `text`：指定合成文本
- `voice`：指定音色
- `language_type`：合成音频语种（可选值：Chinese、English、German、Italian、Portuguese、Spanish、Japanese、Korean、French、Russian、Auto（默认））
- 合成语音通过返回的 URL 获取，URL 有效期为 24 小时

### 基础调用示例（Python/Java/cURL）

#### Python
```python
# DashScope SDK 版本不低于 1.24.6
import os
import dashscope

# 北京地域 URL，新加坡地域请替换为：https://dashscope-intl.aliyuncs.com/api/v1
dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'

text = "那我来给大家推荐一款T恤，这款呢真的是超级好看，这个颜色呢很显气质，而且呢也是搭配的绝佳单品，大家可以闭眼入，真的是非常好看，对身材的包容性也很好，不管啥身材的宝宝呢，穿上去都是很好看的。推荐宝宝们下单哦。"
# 原 SpeechSynthesizer 接口用法：dashscope.audio.qwen_tts.SpeechSynthesizer.call(...)
response = dashscope.MultiModalConversation.call(
    model="qwen3-tts-flash",
    # 新加坡和北京地域 API Key 不同，获取地址：https://help.aliyun.com/zh/model-studio/get-api-key
    # 未配置环境变量时，直接替换为 api_key = "sk-xxx"
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    text=text,
    voice="Cherry",
    language_type="Chinese",  # 建议与文本语种一致，确保发音自然
    stream=False
)
print(response)
```

### 实时播放示例（Python/Java/cURL）

#### Python
```python
# DashScope SDK 版本不低于 1.24.6
# coding=utf-8
#
# PyAudio 安装说明：
# Apple Mac OS X: brew install portaudio && pip install pyaudio
# Debian/Ubuntu: sudo apt-get install python-pyaudio python3-pyaudio 或 pip install pyaudio
# CentOS: sudo yum install -y portaudio portaudio-devel && pip install pyaudio
# Microsoft Windows: python -m pip install pyaudio

import os
import dashscope
import pyaudio
import time
import base64
import numpy as np

# 北京地域 URL，新加坡地域请替换为：https://dashscope-intl.aliyuncs.com/api/v1
dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'

p = pyaudio.PyAudio()
# 创建音频流
stream = p.open(format=pyaudio.paInt16,
                channels=1,
                rate=24000,
                output=True)

text = "你好啊，我是通义千问"
response = dashscope.MultiModalConversation.call(
    # 新加坡和北京地域 API Key 不同，获取地址：https://help.aliyun.com/zh/model-studio/get-api-key
    # 未配置环境变量时，直接替换为 api_key = "sk-xxx"
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    model="qwen3-tts-flash",
    text=text,
    voice="Cherry",
    language_type="Chinese",  # 建议与文本语种一致，确保发音自然
    stream=True
)

for chunk in response:
    if chunk.output is not None:
        audio = chunk.output.audio
        if audio.data is not None:
            wav_bytes = base64.b64decode(audio.data)
            audio_np = np.frombuffer(wav_bytes, dtype=np.int16)
            # 实时播放音频
            stream.write(audio_np.tobytes())
        if chunk.output.finish_reason == "stop":
            print("播放完成时间: {} ", chunk.output.audio.expires_at)

time.sleep(0.8)
# 释放资源
stream.stop_stream()
stream.close()
p.terminate()
```
